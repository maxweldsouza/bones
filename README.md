### Bones

This is a skeleton react project.  
It uses:  

Babel
Eslint
Flow
Jest
PostCss
Sass
Css Modules
Webpack 2
Redux
Immutable.js

To run:  
`yarn install`  
`npm start`  

Go to:  
`localhost:8000`
